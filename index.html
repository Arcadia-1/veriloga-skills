veriloga-skills

A standalone Verilog-A language authoring skill, with optional companion skills kept independent.

Core Skill

veriloga is self-contained. It covers Verilog-A source-language syntax and semantics for .va files: module interfaces, disciplines, analog blocks, branch access, contributions, events, parameters, arrays, operators, and small language examples.

Copying only veriloga/ is enough to install the authoring skill. It does not require local simulation tools, Python packages, project datasets, or the sibling skills.

Skill Contents

References

  • modules, ports, disciplines
  • analog blocks and contributions
  • events, state, and control flow
  • operators and system tasks

Assets

  • neutral module template
  • minimal contribution syntax
  • declaration and parameter syntax
  • cross-event syntax
  • transition syntax
  • vector and genvar syntax
  • named branch access syntax

Install The Authoring Skill

The clone command below is only a copy/install method. It is not a runtime dependency of the veriloga skill after the folder has been copied.

git clone --depth 1 https://github.com/Arcadia-1/veriloga-skills /tmp/veriloga-skills
mkdir -p .agents/skills
cp -r /tmp/veriloga-skills/veriloga .agents/skills/
